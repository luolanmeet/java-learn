### 工厂模式

### 简单概念
* 简单工厂模式：产品的工厂
* 工厂方法模式：工厂的工厂
* 抽象工厂模式：系列产品的工厂

### 案例

| 例子                              | 类型         |
| --------------------------------- | ------------ |
| `LoggerFactory.getLogger("cck");` | 简单工厂模式 |
| `Calendar.getInstance();`         | 简单工厂模式 |
| 数据库连接池                      | 工厂模式     |
